- Case study Google Doc: https://docs.google.com/document/d/1fdwziOr7TuxJrxk4PdVSGU6-fVhdbDhj/edit

**Module breakdown:**
- Module 1: Accordion (4 criteria) + Knowledge Check 5 MCQ
- Module 2: Timeline (5-phase framework) + Checkbox list + Reflective scenario
- Module 3: Accordion (dos/don'ts) + Tabs (before/after) + Sorting activity

**Key design decisions — defensible trong interview:**
1. Reflective scenario thay vì situational judgment test (Module 2) — target audience là experienced teachers, not novices. Right/wrong format signals compliance; reflective format signals professional judgment.
2. Discrimination activity làm terminal assessment (Module 3) — pattern recognition phải có trước production. Consistent với worked example → discrimination → production sequence.
3. Bloom's Taxonomy as cross-module arc — Module 1 (Remember/Understand) → Module 2 (Apply) → Module 3 (Analyse/Evaluate). Vertical coherence within modules + horizontal coherence across course.

**CV Line (Rise 360):**
