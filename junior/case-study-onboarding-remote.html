Case Study 3

IELTS Teacher Onboarding — Rise 360 Micro-learning

Designing a self-paced onboarding ecosystem for distributed academic teams.

Articulate Rise 360 Moodle 4.x SCORM 1.2 Gagne's 9 Events

The Problem

When new IELTS teachers join a center, getting them aligned on marking criteria, lesson structure, and feedback protocols takes time — and usually requires a senior teacher to walk them through everything in person.

The goal of this project was to design a self-paced onboarding experience that new teachers could complete independently before their first class, without needing synchronous handholding.

Core Objective

Create a 3-module Rise 360 course covering Band Descriptors, lesson frameworks, and feedback protocols, deployed as SCORM 1.2 to Moodle.

What I Built

A 3-module self-paced onboarding course in Articulate Rise 360 with:

  • → Mobile-first design — all blocks tested for responsiveness across screen sizes
  • → Gagne's Nine Events of Instruction used as the structural framework for each module
  • → Interactive blocks — labeled graphics, sorting activities, and knowledge checks embedded throughout
  • → SCORM 1.2 export deployed to Moodle with completion tracking and activity gating
  • → Content Aligned to Official Descriptors — mapped directly to source material
Tools Used

Articulate Rise 360 · Moodle 4.x · SCORM 1.2 · IELTS Band Descriptors (official source)

Instructional Flow

The course was built using a minimalist aesthetic to reduce cognitive load, focusing on high-impact interactions that simulate classroom scenarios.

Visual Minimalism: High-impact cover design focusing on core teacher competencies.
Learning Framework

Sequential progression: Understanding the Task → Lesson Framework → Feedback Protocols.

Visual Hierarchy

Block-based structure ensures clarity across desktop and mobile devices.

Design Decisions

Why Rise 360?

Rise 360 is faster to build and easier to update — critical for onboarding content that evolves as criteria changes.

Why Embedded Checks?

Short checks after each section keep adult learners active, rather than passively reading a single long quiz.

Why Official Descriptors?

Using the official source ensures new teachers match their mental model to what examiners actually use, leaving no room for interpretation.

Micro-interactions

Each module integrates interactive elements followed by formative assessments to ensure teachers can apply descriptors to actual student work.

Activity: Deconstructing Band Descriptors through interactive labeled graphics.
Mobile Mastery: Ensuring a "zero-friction" experience for teachers on the move.

What I Learned

  • → Rise 360's block-based structure encourages modular chunking — each block must serve one clear learning objective.
  • → Gagne's Nine Events provide a useful framework for adult learners who need context and relevance before instruction.
  • → Deploying to Moodle via SCORM requires testing completion triggers carefully — defining "complete" must happen before export.
