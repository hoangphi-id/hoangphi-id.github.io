# 📁 JDS — Raw Job Descriptions Archive

Mỗi khi tìm được JD mới, bản gốc (raw text) được lưu ở đây.

## Naming Convention
```
{company-slug}-{role-slug}-{YYYY-MM-DD}.md
```
Ví dụ: `articulate-csm-2026-04-11.md`

## Vì sao lưu JD gốc?
1. JD trên LinkedIn có thể bị xóa bất cứ lúc nào — lưu lại để còn tham khảo
2. Khi cần so sánh nhiều JD với nhau
3. Khi cần tạo lại Battle Card với góc nhìn mới
