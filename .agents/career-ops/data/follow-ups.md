## Status Legend
| Status | Meaning |
|---|---|
| `Sent` | Message sent, no reply yet |
| `Replied` | Contact replied |
| `Ghosted` | No reply after 14+ days |
| `Converted` | Led to interview or application |
| `Closed` | No longer relevant |

## Follow-Up Cadence Rules (per `modes/followup.md`)
- Day 0: Send initial message
- Day 7: One polite follow-up if no reply
- Day 14: Final nudge or mark Ghosted
- Never: More than 2 follow-ups to same person
