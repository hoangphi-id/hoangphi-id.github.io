# 📁 REPORTS — Battle Card Repository

Mỗi khi đánh giá 1 JD, file Battle Card được lưu ở đây.

## Naming Convention
```
{###}-{company-slug}-{YYYY-MM-DD}.md
```
Ví dụ: `001-articulate-2026-04-11.md`

## Cách dùng
- Mở bất kỳ file nào để xem lại phân tích đầy đủ (Blocks A–G)
- So sánh nhiều reports để thấy pattern (dùng `data/applications.md`)
