## ⚠️ CẢNH BÁO KỸ THUẬT: PDF GENERATION BLOCKER
Hiện tại, Antigravity không tìm thấy **Node.js** và **Playwright** trên hệ thống của bạn để chạy script tự động tạo PDF.

**Cách khắc phục:**
1. **Option A (Ưu tiên)**: Bạn cài đặt Node.js (tải từ nodejs.org) và chạy lệnh:
   `npm install playwright && npx playwright install chromium`
2. **Option B (Thủ công)**: Mở file `.html` (khi được tạo) bằng Google Chrome và chọn `Print -> Save as PDF`. Layout đã được tối ưu cho Print.
